Executive Context

Medical imaging hardware advances rapidly, creating pressure on healthcare centers with 10–15 year equipment lifecycles. While OEM SaaS solutions exist, vendor lock-in and varying quality remain challenges. Our OEM-agnostic GenAI SaaS offers a path to upgrade image quality without immediate capital expenditure, providing flexibility across equipment vendors and imaging protocols.

Developer Documentation

Integrate our GenAI enhancement API into your existing workflow

REST API

Submit Enhancement Job

POST /v1/enhance
Content-Type: multipart/form-data

{
  "modality": "XRAY",
  "studyId": "study_12345",
  "enhancement": {
    "superResolution": true,
    "denoising": true,
    "strength": 0.8
  }
}

Check Job Status

GET /v1/jobs/job_67890

Response:
{
  "jobId": "job_67890",
  "status": "completed",
  "progress": 100,
  "resultUrl": "https://api.agnosticimagingai.com/results/enhanced_study_12345.dcm",
  "qualityMetrics": {
    "confidence": 0.95,
    "enhancement_factor": 3.2
  }
}

TypeScript SDK

import { AgnosticImagingAI } from '@agnosticimagingai/sdk';

const client = new AgnosticImagingAI({
  apiKey: process.env.AGNOSTIC_API_KEY
});

const job = await client.enhance({
  file: dicomFile,
  modality: 'XRAY',
  options: { strength: 0.8 }
});

const result = await job.waitForCompletion();

cURL Example

curl -X POST "https://api.agnosticimagingai.com/v1/enhance" \
  -H "Authorization: Bearer $API_KEY" \
  -H "Content-Type: multipart/form-data" \
  -F "file=@xray_image.dcm" \
  -F "modality=XRAY" \
  -F "studyId=study_12345"
